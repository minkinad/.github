# Community Health Files

Default community health files for repositories under [minkinad](https://github.com/minkinad).

This repository contains the shared files GitHub can reuse across projects when a repository does not define its own community files.

- Contribution guide: [minkinad/contribute](https://github.com/minkinad/contribute)
- GitHub profile: [@minkinad](https://github.com/minkinad)
- Website: [minkin.tech](https://minkin.tech)

Most repositories here are experiments, tools, and product work around TypeScript, frontend engineering, Node.js, parsing, and developer tooling.

If you want to contribute, start with the shared guide above.
